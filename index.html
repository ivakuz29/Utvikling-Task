How the concept works

This is the flow the final product could follow once you add JavaScript and a backend later.

1

Collect prices

Store prices are captured from feeds, manual input, or future scraping pipelines.

2

Compare changes

The system compares current price against previous price to find sudden drops.

3

Match by location

Users can set a home area so they only see deals from stores that are actually close.

4

Send alerts

When a tracked item gets cheaper nearby, users receive a notification instantly.
