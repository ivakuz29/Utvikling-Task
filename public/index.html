The process

From raw data
to alert in four steps.

Price snapshots are recorded from stores via feeds or manual submissions.

2

Compare changes

Each new price is compared against the previous entry to detect drops.

3

Match by location

Users set their area so they only see deals from nearby stores.

4

Send alerts

When a tracked item gets cheaper nearby, a notification is dispatched instantly.
